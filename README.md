# resumo-do-lab
Vi que existem 3 tipos de computação em nuvem
1 nuvem private: Onprimese
2 Nuvem pública: Azure, AWS, Google cloud
3 Nuvem hibrida: a combinação de onPrimese + Nuvem pública
A nuvem pública, em resumo, é a virtualização de tudo que temos no ambiente onPrimese
Existem diversos recursos na nuvem pública:
servidores virtuais;
clusters;
recursos de rede;
bancos de dados;
firewall;
balanceadores de cargas;
recursos de IA;
